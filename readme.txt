This program reads in user input via the keyboard.

to run the program, open terminal and navigate to the directory that
contains the ASN01 file that is found on the project disk.

simply type in ./ASN01 to start the program.

NOTE1: permissions may not be set
if running into permission denied, type in "chmod 777 ASN01" without
the quotation marks.

NOTE2: user must be using linux

Commands include:
ctrl + k: sends an interupt that exits the program
T: terminates the program
E: sends input to translate
X: backspace (only occurs in translate process)
K: kill line (only occurs in translate process)
